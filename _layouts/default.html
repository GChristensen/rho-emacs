RHO Emacs

Download installer for Windows (64-bit)

SEE ALSO: Enso open-source, Scrapyard

Description

ρEmacs is a preconfigured distribution of GNU Emacs editor for Microsoft Windows. It offers some enhancements over the standard Emacs experience and targets several Windows-specific issues. Additionally, it provides optional components which improve org-mode capture capabilities and allow to create an org-mode-based wiki.

Features

  • A smart launcher, that starts Emacs in a server mode and then acts like emacsclient.
  • The home directory of the standard Emacs is bound to USERPROFILE or HOME environment variables. The home directory of ρEmacs could be anywhere, and even could be specified as a command-line argument to the launcher.
  • ">œÅEmacs includes several popular libraries, modes and themes which comprise a reasonably convenient configuration out of the box.
  • Some common POSIX utilities used by Emacs, such as find/grep, patch/diff and aspell, are provided by the distribution.
  • Capture with org-protocol may be configured by a single check in the installer.
  • Possibility of a portable installation.
