RHO Emacs

Download installer for Windows (64-bit)

SEE ALSO: Enso open-source, Scrapyard

Description

ρEmacs is a preconfigured distribution of GNU Emacs editor for Microsoft Windows. It offers some enhancements over the standard Emacs experience and targets several Windows-specific issues. Additionally, it provides optional components which improve org-mode capture capabilities and allow to create an org-mode-based local wiki.

Features

  • A smart launcher, that starts Emacs in a server mode and then acts like emacsclient.
  • The home directory of the standard Emacs is bound to USERPROFILE or HOME environment variables. The home directory of ρEmacs could be anywhere, and even could be specified as a command-line argument to the launcher.
  • ρEmacs includes several popular libraries, modes and themes which comprise a reasonably convenient configuration out of the box.
  • The Hack typeface, which works quite fast with Emacs font-lock mode is used by default. This helps to avoid lags and freezes when editing large files.
  • Some common POSIX utilities used by Emacs, such as find/grep, patch/diff and aspell, are also provided by the distribution.
  • Capture with org-protocol may be configured by a single check in the installer.
  • Possibility of a portable installation.

ρEmacs launcher

ρEmacs could be launched using a shortcut from Windows Start Menu, but the launcher is also accessible at the command line as the rho command (included in system PATH environment variable). It passes any arguments that do not start with a slash (/) to the underlying program which may be emacs if Emacs is not running, or emacsclient otherwise. The /HOME command-line argument allows to specify the home directory ρEmacs in the form /HOME:path/to/the/directory. This is convenient, for example, for use in Windows shortcuts.

Use rhoc command to run Emacs in the terminal (it will act as emacsclient if Emacs is already running.

ρEmacs enhancement libraries

ido, fido, iswitchb, ibuffer, uniquify, save-place, desktop-save, org-bullets, tabbar and bookmark+ modes could be enabled by default.
